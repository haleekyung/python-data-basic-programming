  * 공부의 양과 상관없이 최소 하루 잔디 1개를 채울 것

  * 경고 2회 누적시 공부의지가 없는 것으로 간주하여 최종 패널티 실행하며, 검사시간은 매주 금 또는 토요일로 정한다.
